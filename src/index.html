お知らせNews

  • 2020.01.01
    Info

    当サイトはORイケダによる制作サンプルサイトです。

    • 「お知らせ」欄を除く全ての内容は架空のものとなっております。
    • トップページのみの見本として作成しており、下層ページは存在しません。
    • ページ上部にある文字サイズ変更ボタンは現時点では機能しません。近日中に実装予定です。
    • コードはGitHubでご覧いただけます。
  • 2020.5.23
    Code

    CSSの記述方式をFLOCSSを意識したスタイルに変更しました。また、HTMLやJavaScriptの記述も見直しと修正を行ました。

  • 2019.12.2
    Code

    モバイルデバイスへのレイアウト対応を完了しました。画面幅959px以下はタブレットに、また559px以下はスマートフォンに最適化したデザインとなります。

  • 2019.11.18
    Note

    グローバルナビゲーション部分（ページ上部の「企業情報」などが表示されているパート）のドロップダウンメニュー化を完了しました。これにより、サブメニューが表示されるようになりました。jQueryのプラグインなどは使用せず、CSSのみで実装しています。

  • 2019.11.17
    Code

    メインビジュアルにスライドショーを実装しました。プラグインは使用せず、シンプルなJavaScriptで記述しました。

  • 2019.11.16
    Code

    当サイトは、現時点ではMicrosoft社製のブラウザ「Internet Explorer」には対応しておりません。「Internet Explorer」で閲覧した場合、レイアウトの崩れが生じるなど、快適なブラウジングが出来ない場合があります。

  • 2019.11.16
    Note

    当サイトでは、バナーを含む全ての画像について、素材サイトfreepikのものを使用しております。当事務所はfreepikの正規ユーザであるため、画像利用に関するクレジットは表記しておりません。
