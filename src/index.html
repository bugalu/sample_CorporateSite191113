  • 2019.11.18
    Info

    当サイトはORイケダ事務所による制作サンプルサイトです。「お知らせ」欄を除く全ての内容は架空のものとなっております。

  • 2019.11.18
    Note

    グローバルナビゲーション部分（ページ上部の「企業情報」などが表示されているパート）のドロップダウンメニュー化を完了しました。これにより、サブメニューが表示されるようになっております。実装には、jQueryのプラグインなどは使用せず、CSSのみで記述いたしました。

  • 2019.11.17
    Code

    スライドショー（カルーセル）を実装しました。プラグインは使用せず、シンプルなJavaScriptで記述しております。

  • 2019.11.17
    Code

    現時点ではレスポンシブ対応コーディング（スマートフォンやタブレット端末へ最適化したデザイン処理）を行っておりません。そのため、スマートフォンなどの小さな画面での閲覧には適しておりません。（数日内に対応させる予定となっております。）

  • 2019.01.16
    Code

    当サイトは、Microsoft社製のブラウザ「Internet Explorer」には対応しておりません。「Internet Explorer」で閲覧した場合、レイアウトの崩れが生じるなど、快適なブラウジングが出来ない場合があります。

  • 2019.11.16
    Note

    当サイトでは、バナーを含む全ての画像について、素材サイトfreepikのものを使用しております。当事務所はfreepikの正規ユーザであるため、画像にはクレジットを表記しておりません。
